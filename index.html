Passo a Passo para Iniciantes

Passo 1: Abrir o Console do Navegador

Para abrir o console do navegador, geralmente você pode fazer isso pressionando F12 ou Ctrl+Shift+I (ou Cmd+Option+I no macOS). Também é possível clicar com o botão direito do mouse e procurar por "Inspecionar". Isso abrirá as Ferramentas de Desenvolvedor.

Procure pela aba Console.

Passo 2: Escrever e Executar "Hello, World!"

No console aberto, você verá um prompt onde pode digitar código JavaScript. Digite o seguinte código:

console.log("Hello, World!");

Isso instrui o navegador a imprimir "Hello, World!".

Após digitar o código, pressione Enter. Você verá imediatamente a mensagem "Hello, World!" aparecer no console.
